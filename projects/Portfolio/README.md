# Ibrahim Abdullah - Personal Portfolio

Welcome to the repository of my personal portfolio website! This project showcases my skills, projects, certifications, and coding activities as a Frontend Developer / Software Engineer.

## 🚀 Live Demo
[View Live Portfolio](https://ibraheamit.github.io/frontend-engineer)

## ✨ Features
- **Modern UI/UX**: Dark theme with beautiful CSS gradients, glassmorphism, and smooth scroll animations.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
- **Dynamic Projects Showcase**: A dedicated projects page (`projects.html`) rendering items dynamically via JavaScript.
- **Activities & Achievements**: Sections tracking progress on platforms like LeetCode and Codewars (`activities.html`).
- **Certifications Gallery**: Highlighting professional training like DEPI and ALX (`certifications.html`).
- **Contact Section**: An interactive section for collaboration inquiries.

## 🛠️ Tech Stack
- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom properties (variables), Flexbox, CSS Grid, animations, and media queries.
- **JavaScript (Vanilla)**: DOM manipulation, event handling, dynamic grid rendering, and interactive modals.

## 📂 Project Structure
```text
├── assets/
│   ├── img/            # Images (projects, certifications, logos, etc.)
├── index.html          # Main landing page
├── projects.html       # Detailed projects showcase page
├── activities.html     # Coding activities, stats, and hackathons
├── certifications.html # Certificates and professional achievements
└── README.md           # This documentation file
```

## 💻 How to Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/ibraheamit/frontend-engineer.git
   ```
2. Navigate to the project directory:
   ```bash
   cd frontend-engineer
   ```
3. Open `index.html` in your favorite web browser. Since it's built with Vanilla HTML/CSS/JS, no build steps are required!
   *(Optional: You can use an extension like VS Code Live Server for auto-reloading during development).*
